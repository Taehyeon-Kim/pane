# Homebrew Tap for Pane

This is the official Homebrew tap for [Pane](https://github.com/Taehyeon-Kim/pane), a blazing-fast TUI skill launcher for developers.

## Installation

Add the tap and install Pane:

```bash
brew tap Taehyeon-Kim/pane
brew install pane
```

Or install directly without tapping:

```bash
brew install Taehyeon-Kim/pane/pane
```

## Usage

After installation, you can run Pane from your terminal:

```bash
pane
```

## Updating

To update to the latest version:

```bash
brew upgrade pane
```

## Uninstalling

To uninstall Pane:

```bash
brew uninstall pane
```
